How

This website was built from the ground up using HTML, CSS, and JavaScript; intentionally keeping things simple and lightweight. No frameworks, no tracking scripts, no unnecessary complications. It is currently hosted on GitHub and the domain was acquired through Porkbun.

True to the principles outlined in the 'Why' section, this site respects your privacy. There are no analytics, no cookies (except for saving your theme preference locally), and no data collection of any kind. The only JavaScript here handles the dark/light mode toggle, tag filtering for updates, and basic page functionality. Even the theme preference is stored in your browser's localStorage rather than cookies, keeping things privacy-friendly.

The only external resources loaded are the fonts and icons needed to make the site functional and readable. Everything else is plain, vanilla code with no frameworks or third-party services tracking your activity.

If there are any significant changes in the way this site works, you'll find out through this page or the Changelog.
